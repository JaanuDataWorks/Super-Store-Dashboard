# 📊 Global Store Dashboard – Performance Analysis

## 🧾 Overview
This project is a Power BI dashboard that analyzes the performance of a retail store using key business metrics such as sales, profit, orders, and delivery time.

## 🎯 Objectives
- Analyze sales and profit across different states
- Track customer orders and quantity sold
- Evaluate delivery lead time and shipping cost
- Identify high-performing and low-performing regions

## 📌 Key Insights
- New York generates the highest sales
- Some regions have high shipping cost affecting profit
- Delivery time varies across states impacting efficiency

## 🛠️ Tools Used
- Power BI
- DAX
- Power Query
